# Обзор приложения

Персональный таплинк (link-in-bio) для акушера-гинеколога (Кристина Кургинян, Москва). Одностраничный сайт с длинным скроллом, мобильный приоритет, два языка (RU/EN). Главная задача — превратить переход из профиля в соцсети в обращение и запись на приём.

## Что внутри

Одна страница, собранная из секций (сверху вниз):

| Секция | Файл | Назначение |
|--------|------|-----------|
| Шапка | `Header.tsx` | Лого-монограмма, навигация (десктоп), переключатель языка. Прозрачная, затемняется при скролле. |
| Hero | `Hero.tsx` | Имя, специальность, краткое описание, портрет (`Portrait.tsx`), две CTA-кнопки, чипы. |
| Услуги | `Services.tsx` | Горизонтальный слайдер направлений работы (scroll-snap, стрелки, точки). |
| Цены | `Prices.tsx` | Прайс-лист приёмов: услуга, описание, стоимость. Цены ориентировочные. |
| О враче | `About.tsx` | Текст + таймлайн образования и опыта. |
| Сертификаты | `Certificates.tsx` | Слайдер сертификатов: текст либо текст + фото (открывается по клику в лайтбоксе). |
| Отзывы | `Reviews.tsx` | Слайдер отзывов со звёздным рейтингом и аватаром (фото или заглушка, фото открывается по клику). Тексты-заглушки. |
| Вопросы | `Faq.tsx` | Заголовок сверху, список вопросов на всю ширину; аккордеон на нативном `<details>`. |
| Запись | `Contact.tsx` | Тёмная карточка с CTA и каналами связи (WhatsApp, телефон, email, город). |
| Подвал | `Footer.tsx` | Компактная строка: специальность · город и копирайт. |
| Кнопка записи | `StickyCta.tsx` | На мобильных — нижняя панель, на десктопе — круглая иконка-кнопка в правом нижнем углу (якорь на запись). |

Общие переиспользуемые части: `useSlider.ts` (логика слайдеров), `Slider.tsx` (стрелки/точки), `Lightbox.tsx` (просмотр картинок), `Avatar.tsx` (аватар с заглушкой), `TruncatedText.tsx` (обрезка текста троеточием + кастомная подсказка, которая позиционируется вверх/вниз/в сторону и не выходит за экран).

## Технологии

- **React 19 + TypeScript + Vite** — каркас и сборка.
- **framer-motion** — анимации появления секций, hero, слайдов.
- **lenis** — плавный инерционный скролл.
- **react-i18next + i18next** — переключение языков RU/EN.
- **lucide-react** — иконки.
- **CSS** — токены дизайна в `src/index.css`, стили секций в `src/styles/sections.css` (OKLCH, fluid `clamp()`).

## Дизайн

Направление — «тёплый кабинет частной клиники»: глубокий чернильный индиго несёт премиальность и доверие, тёплое латунное золото добавляет тепло, чистый светлый фон для чтения. Тепло и деликатность, уместные для акушера-гинеколога, идут от цвета, типографики и фото, а не от клише. Подробности в [DESIGN.md](./DESIGN.md), стратегия и аудитория — в [PRODUCT.md](./PRODUCT.md).

Навигация: все переходы по якорям (меню, кнопки, «листайте вниз») плавные — перехватываются в `useSmoothScroll.ts` и прокручиваются через Lenis с отступом под шапку.

Доступность: контраст WCAG AA, видимый фокус, поддержка `prefers-reduced-motion` (анимации заменяются мягким фейдом, плавный скролл отключается и переходит в мгновенный с тем же отступом), тач-таргеты ≥44px.

## Запуск

```bash
cd app/front
npm install
npm run dev      # дев-сервер (http://localhost:5173)
npm run build    # продакшн-сборка в app/front/dist
npm run preview  # предпросмотр сборки
```

## Что нужно заменить перед публикацией

- **Образование, опыт, ординатура** (`about.credentials` в `content/`) — сейчас это заглушки-подсказки («Укажите вуз…»). Впишите реальные данные акушера-гинеколога.
- **Сертификаты** (`about.certificates`) — примерный список, замените реальными сертификатами и датами.
- **Отзывы** (`reviews.items`) — тексты и имена сейчас образцы-заглушки. Замените настоящими отзывами пациенток (с их согласия). Не публикуйте вымышленные отзывы как реальные.
- **Цены** (`prices.items`) — суммы ориентировочные. Проставьте актуальную стоимость приёмов.
- **Портрет** — `public/portrait.png` сейчас сгенерированная заглушка (не реальное фото). Замените на настоящее фото Кристины. Если файла нет или он не загрузился, автоматически показывается абстрактная заглушка с монограммой (`Portrait.tsx`).
- **Кнопка записи** — ведёт на WhatsApp. При желании подключите систему записи (Yclients, форму и т.д.) в `Contact.tsx` и `Hero.tsx`.
- **Тексты и контакты** — правятся в `src/content/ru.ts` и `src/content/en.ts` (см. [DATA.md](./DATA.md)). Телефон, WhatsApp и email подставлены из резюме.
- **Монограмма и favicon** — `public/favicon.svg`; инициалы в шапке выводятся из имени автоматически.

> Важно: специальность (акушер-гинеколог) задана по требованию заказчика и отличается от специальности в исходном резюме (педиатр). Поэтому все профессиональные данные (образование, опыт, сертификаты) требуют подтверждения реальными сведениями перед публикацией.
