# Taplink — персональный сайт врача

Персональный таплинк (link-in-bio) для врача-педиатра (Кристина Кургинян, Москва): одна страница с записью на приём, информацией о враче, направлениями помощи, навыками, опытом и частыми вопросами. Двуязычный (RU/EN), мобильный приоритет, плавные анимации и переходы.

![Стек](https://img.shields.io/badge/React-19-1b1c3a) ![Vite](https://img.shields.io/badge/Vite-8-1b1c3a) ![TypeScript](https://img.shields.io/badge/TypeScript-strict-1b1c3a)

## Структура репозитория

```
taplink/
├─ app/
│  ├─ front/        # фронтенд (React + Vite + TS) — всё приложение
│  └─ docs/         # документация
│     ├─ OVERVIEW.md   # обзор приложения, секции, запуск
│     ├─ DATA.md       # модель контента и как его править
│     ├─ PRODUCT.md    # продуктовый контекст: аудитория, бренд, принципы
│     └─ DESIGN.md     # дизайн-система: цвета, типографика, моушн
├─ README.md        # этот файл
└─ AGENTS.md        # инструкции для AI-агентов
```

## Быстрый старт

```bash
cd app/front
npm install
npm run dev
```

Откройте `http://localhost:5173`.

| Команда | Действие |
|---------|----------|
| `npm run dev` | Дев-сервер с HMR |
| `npm run build` | Продакшн-сборка в `app/front/dist` |
| `npm run preview` | Локальный предпросмотр сборки |
| `npm run lint` | Проверка линтером (oxlint) |

## Возможности

- **Запись на приём** — основная цель страницы, кнопка доступна везде (hero, плавающая кнопка на мобильных, секция контактов).
- **Два языка (RU/EN)** — переключатель в шапке, выбор сохраняется в браузере.
- **Премиальный дизайн** — глубокий индиго + латунное золото, сериф `Spectral` + гротеск `Golos Text`.
- **Анимации** — появление секций (`framer-motion`), плавный скролл (`lenis`), слайдер направлений.
- **Доступность** — контраст WCAG AA, фокус с клавиатуры, поддержка `prefers-reduced-motion`.

## Что заменить перед публикацией

Контент и контакты редактируются без правки вёрстки — см. [app/docs/DATA.md](app/docs/DATA.md). Кратко:

- тексты и контакты — `app/front/src/content/ru.ts` и `en.ts` (данные взяты из резюме);
- портрет — `app/front/public/portrait.png` (сейчас сгенерированная заглушка, не реальное фото);
- кнопка записи — ведёт на WhatsApp, при желании подключите систему записи;
- цен на странице нет (в резюме нет данных о стоимости) — при необходимости добавьте секцию прайса.
